# Data Science

Esercizi e script di live coding ricavati dal percorso **Master Data Analytics (MDA) Fundamentals** @ Roma Tre

Il master copre 26 corsi in 320 ore su statistica, machine learning, Python, SQL e data visualization. Questo repo raccoglie il materiale pratico: esercizi per consolidare i concetti e script da eseguire per le sessioni di studio. Il materiale in questo repo è la mia personale trasposizione pratica della knowledge base acquisita durante il percorso. 

## Prerequisiti

Il repo assume familiarità con Python (sintassi base, basi OOP, strutture dati)
